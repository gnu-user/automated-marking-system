Assignment 1

Please make sure that you test and evaluate your submission thoroughly before the final submission, your code must not have any compiler errors in order for it to be evaluated. Any submissions that do not compile cannot be evaluated and will receive a grade of zero.

Description

For your assignment you must create a commandline calculator program which will accept any two numbers as input (both integers and floating point values) and perform one of the following requested operations add, subtract, multiply, divide and display the results.

Requirements

  • 60 Maximum execution time (seconds)
  • 1 Maximum attempts
  • C++ Language
  • 10% Code analysis weight
  • 90% Test case weight
